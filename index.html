NUKUNTU

Nukuntu, adalah distro linux yang di custom dari Runtu lite 14.04, dibuat untuk kebutuhan individu penulis. Distro ini Ubuntu Based, dimana paket software dan data masih menginduk ke Ubuntu dan Canonical. Sudah ditambahkan beberapa software tambahan dari software yang sudah tersedia di Runtu itu sendiri. Fast, Small, and Stable seperti itulah ungkapan dari distro Nukuntu ini. Default Desktop Environment memakai LXDE, sesuai dengan default dari Runtu. Cocok untuk user yang berkebutuhan multimedia seperti Nonton Movies, Anime dan Memutar Lagu.
